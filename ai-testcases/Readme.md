# Creating testcases for your course activity

1. Make a pull request
2. Copy the tescase template into the activity subfolder (example: quiz, project) and rename it to reflect what it covers. 
   Example: testcase-quiz2-germancredit-XY.md, where XY is your initial.
3. Fill the testcase. Make sure to add your name under 'TC-identifier:'
4. Commit
5. We will merge your changes.
